# AOC 2025

These solutions are written in [Racket](https://racket-lang.org/).

Require Racket 9.0 and rosette package

# Run

Install `advent-of-code` package:

```shell
raco pkg install advent-of-code
```

Show help messages:

```shell
racket aoc.rkt -h
```

Compile Racket files for faster startup time:

```shell
raco make **/*.rkt
```

# Test

Test for the example and my own input:

```shell
raco test tests
```
